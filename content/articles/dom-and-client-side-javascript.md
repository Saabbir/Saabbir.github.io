### DOM tree structure as tree of nodes

The one at the bottom inherit properties and methods from the top.

- EventTarget
  - Node
  - Document
  - CharacterData
    - Text
    - Comment
  - Element
    - HTMLElement
      - HTMLAnchorElement
      - HTMLInputElement
      - HTMLTableElement
      - ...and many more
