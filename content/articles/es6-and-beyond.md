### The most important features of ES2015

- Block Scoping (Let and Const)
- Template Literals
- Arrow Functions
- Default Parameters
- Rest Parameters
- Spread Syntax
- Enhanced Object Initializer
- Enhanced Object Method Definitions
- Destructuring Assignment
- Classes
- Generator functions
- Promise
